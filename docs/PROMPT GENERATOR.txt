PROMPT GENERATOR
================

# Posibilidad de incluir archivos y directorios

POST /api/v1/prompts

{
    "description":"some text",
    "files":[
        "D:\\laragon\\www\\simplerest\\app\\controllers\\SignatureTestController.php",
        "D:\\Desktop\\PROMPT GENERATOR\\"
    ],
    "notes":""
}

# Recuperacion de prompts

GET /api/v1/prompts?page=1


# Posibilidad de incluir archivos desde peticiones HTTP

Esto es util para archivos como HTML, CSS y Javascript

POST /api/v1/prompts

{
	"title": "Prueba de Generador de prompt para ChatGTP",
	"description": "Analiza estos archivos y dime que hacen",
	"files": [      
        "http://relmotor.lan/wp-content/plugins/relmotor-central/app/shortcodes/relmotor/assets/js/search-engine.js?ver=1.1.151",		
        "D:\\laragon\\www\\simplerest\\app\\controllers\\api\\Prompts.php"
	],
	"notes": "Gracias!"
}

# Forma extendida de incluir archivos

Esta forma habilita funcionalidades de reduccion de codigo, posibilidad de espeecificar que archivos o carpetas incluir o excluir en base a patrones glob, etc

POST /api/v1/prompts

{
    "description": "Intento debuguear una query pero el flag $exec de la clase Model cambia de false a true inesperadamente",
    "files": [
        {
            "path": "D:\\laragon\\www\\simplerest\\app\\core\\Model.php"
        },
        {
            "path": "D:\\laragon\\www\\simplerest\\app\\core\\traits\\QueryBuilderTrait.php"
        },
        {
            "path": "D:\\laragon\\www\\simplerest\\app\\core\\traits\\SubResourceHandler.php"
        },
        {
            "path": "D:\\laragon\\www\\simplerest\\app\\core\\traits\\InsertWithSubResourcesTrait.php"
        },
        {
            "path": "D:\\laragon\\www\\simplerest\\app\\controllers\\TestInsertController.php"
        }
    ],
    "notes": "\n"
}

# Posibilidad de excluir funciones de clases PHP para recucir codigo

{

	"description": "Analiza estos archivos y dime que hacen",
	"files": [
        {
            "path":"D:\\laragon\\www\\simplerest\\app\\core\\libs\\ClaudeAI.php",
            "allowed_functions": [
                "__construct",
                "exec", 
                "exec_messages", 
                "getContent"
            ]
        }
	],
	"notes": "Gracias!",
    "mode_simulate": false
}

# Posibilidad de truncar el body de funciones y metodos

Incluir funciones a truncar

POST /api/v1/prompts

Ej:

{
    "title": "Prueba de Generador de prompt",
    "description": "Analiza estos archivos y dime que hacen",
    "files": [
        {
            "path": "D:\\laragon\\www\\simplerest\\app\\core\\helpers\\db.php",
            "interface_replacement": {
                "include": [ "inSchema" ]
            }            
        },
        {
            "path": "D:\\laragon\\www\\simplerest\\app\\core\\Model.php"
        },        
    ],
    "notes": "Gracias!",
    "mode_simulate": false
}

Excluir funciones / metodos a truncar

Ej:

POST /api/v1/prompts

{
    "description":"Test",
    "files":[
        {
            "path":"C:\\Python310\\Lib\\site-packages\\facebook_page_scraper",
            "exclude": ["*.pyc"]
        }
    ],"notes":""
}

# Deshabilitar el guardado en base de datos

Enviando "mode_simulate" en true

Ej:

{
    "title": "Prueba de Generador de prompt",
    "description": "Analiza estos archivos y dime que hacen",
    "files": [
        {
            "path": "D:\\laragon\\www\\simplerest\\app\\core\\helpers\\db.php",
            "interface_replacement": {
                "include": [ "*" ],
                "exclude": [ "inSchema" ]
            }            
        },
        {
            "path": "D:\\laragon\\www\\simplerest\\app\\core\\Model.php"
        }
    ],
    "notes": "Gracias!",
    "mode_simulate": true
}
