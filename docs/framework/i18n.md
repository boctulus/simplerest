# Internacionalización (i18n) — SimpleRest

## Descripción

Sistema de internacionalización basado en **gettext** con archivos de traducción en `app/Locale/`.

**Archivo**: `src/framework/Libs/i18n/`

---

## Estructura

```
app/Locale/
├── es/           # Español
│   └── LC_MESSAGES/
│       ├── messages.po
│       └── messages.mo
├── en/           # Inglés
│   └── LC_MESSAGES/
│       ├── messages.po
│       └── messages.mo
└── ...
```

## Configuración

```php
// app.php (bootstrap)
// El locale se configura en .env o config
```

## Uso

```php
// En helpers globales
echo __('Welcome to SimpleRest');
echo _e('Hello %s!', $name);

// En validación
Translate::bind('validator');

// En controladores
$greeting = trans('greeting.morning');
```

## Dominios de Traducción

| Dominio | Ámbito |
|---------|--------|
| `messages` | Mensajes generales |
| `validator` | Mensajes de validación |
| `acl` | Mensajes del sistema ACL |
