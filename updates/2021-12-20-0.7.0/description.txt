UPDATE DESCRIPTION
==================

- Generalización del "UPDATE sobre sub-recursos" que permite editar campos de estos y no solo las relaciones.

- Contenedor de dependencias inversas experimental.

- Ahora a nivel de modelo se introdujeron más funcionalidades relacionadas con Soft Delete.

    withTrashed(bool $soft_delete)	borra un registro. Hay parámetros adicionales.
	trashed()       				devuelve si el registro fue borrado suavemente.
    undelete()      				restaura el registro si fue borrado suavemente.
    forceDelete()   				borra un registro definitvamente.
    withTrashed()   				incorpora a la selección registros borrados suavemente.
    onlyTrashed()   				selecciona solo registros borrados suavemente.vemente

- Nuevos eventos sobre el modelo:

    onRestoring()
    onRestored()

- Nuevas opciones en comandos "make"

    Ahora es posible borrar una interface / lib / helper / etc con el mismo comando con que se creó. Ej:

    php com make interface gol --remove
